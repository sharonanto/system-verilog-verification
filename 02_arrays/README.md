# Arrays in SystemVerilog

This section contains array-based SystemVerilog examples used for practicing data storage, iteration, and mathematical operations.

## Examples Included

### 1. Scaled Average Computation
- Initializes a byte array
- Computes average value
- Applies scaling factor
- Converts real value to integer

### 2. Square Array Generation
- Stores square values in an array
- Uses loops for initialization
- Displays indexed array elements

## Concepts Practiced
- Static arrays
- for loops
- Arithmetic operations
- Array initialization
- Data display formatting
- Type conversion
